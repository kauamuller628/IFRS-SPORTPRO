# SPORTPRO – E-commerce de Artigos Esportivos

**SPORTPRO** é um projeto de e-commerce **front-end** que simula uma loja virtual de artigos esportivos. O sistema permite navegar por produtos, aplicar filtros, visualizar detalhes, escolher variações e gerenciar um carrinho de compras, tudo sem backend.

## Objetivo
Demonstrar conceitos essenciais de desenvolvimento web utilizando **JavaScript puro**, com foco em organização de código, lógica clara e boa experiência do usuário.

## Público-Alvo
- Consumidores de artigos esportivos  
- Atletas amadores e profissionais  
- Estudantes e desenvolvedores iniciantes  
- Avaliações acadêmicas de front-end  

## Principais Funcionalidades
- Listagem de produtos com filtros (categoria, preço e ordenação)  
- Página de detalhes com seleção de tamanho, cor e quantidade  
- Carrinho de compras funcional  
- Atualização e remoção de itens  
- Cálculo automático de valores  
- Persistência de dados com **LocalStorage**

## Tecnologias Utilizadas
- HTML5  
- Tailwind CSS (CDN)  
- JavaScript (ES6+)  
- LocalStorage
